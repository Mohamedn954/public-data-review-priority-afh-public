# wa_docs/ — Which File Is Canonical

This folder contains an earlier and a later version of three documents, kept
side by side for provenance rather than deleted. The later version in each
pair is canonical; the earlier version is retained for transparency about how
the evidence base evolved, not as an equally-valid alternative.

| Canonical (use this) | Superseded (kept for provenance) |
|---|---|
| `00_README_Extended_WA_Evidence_Base.md` | `00_README_WA_Evidence_Base.md` |
| `WA_Data_Availability_Matrix_Updated.md` | `WA_Data_Availability_Matrix.md` |
| `WA_Expanded_Bibliography.md` | `WA_Source_Bibliography.md` |

The canonical status above is confirmed by `00_README_Extended_WA_Evidence_Base.md`
itself (the "Final Deliverable Package"), which lists `WA_Data_Availability_Matrix_Updated.pdf`
and `WA_Expanded_Bibliography.pdf` as its own deliverables — the superseded
files are not referenced by the final package.

All other files in this folder (`WA_Audit_Oversight_Findings.md`,
`WA_MN_Comparison_Matrix.md`, `WA_Readiness_Memo.md`,
`WA_RedFlag_Testability_Memo.md`) have no duplicate and are canonical as-is.

## Archived (do not use operationally)

`archive/WA_Public_Data_Review_Priority_Tier.md` (and its `.pdf`) contains an
early, never-adopted 0-100 point scoring rubric. It has been moved out of the
canonical set and given a prominent archived-document warning banner, because
a numeric point rubric like this is easy to misread, out of context, as an
actual risk score assigned to a real facility. It is not: the rubric was
never statistically validated, never applied to any real facility, and is not
used anywhere in the manuscript, the anonymized dataset, or the pipeline
scripts. The manuscript's Appendix B (qualitative PDRT framing) is the
canonical, publication-facing version of this idea, and deliberately avoids
assigning any single composite numeric score for exactly this reason.

**Framing note.** Everything in this folder is supporting evidence-base
material — working memos written during the project's exploratory phase, one
step removed from the publication-facing framing. For the canonical framing
of what this project does and does not claim, see the manuscript and
`GUARDRAILS.md` at the repository root; where a memo in this folder differs
from those documents in tone or claim strength, the manuscript and
`GUARDRAILS.md` control. Several memos in this folder carry their own
`Status note` documenting language that was revised after an early draft
used stronger claims than the evidence supported.
