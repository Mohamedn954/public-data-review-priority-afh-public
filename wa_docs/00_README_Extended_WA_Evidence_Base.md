# Washington State AFH/HCBS Program Integrity Evidence Base (Extended)
**Final Deliverable Package**

This package extends the Washington State Adult Family Home (AFH) evidence base to 10 deliverables. It tests 20 review-priority indicators, derived from Minnesota's public Medicaid enforcement record, against Washington public data, using verifiable sources only. No fraud is alleged against any provider.

## Directory of Deliverables

### Datasets & Visualizations
1. **`WA_AFH_Growth_2013_2026.csv` / `.png`** (Section A)
   - Reconstructed year-by-year AFH count, highlighting the post-2018 surge.
2. **`WA_Medicaid_AFH_Contract_Trends_HISTORICAL.csv`** (Section B)
   - Official 2009-2018 Medicaid-contracted facility and bed counts.
3. **`WA_AFH_Enforcement_2020_2026.csv`** (Section C)
   - Facility-level enforcement summary for King/Pierce/Spokane. *Note: 2020-2022 history is not publicly retained by the DSHS portal.*
4. **`WA_County_Normalized_Risk.csv`** (Section D)
   - AFH density and enforcement rates normalized against the Census 65+ population for 9 target counties.

### Memos & Analysis Documents
5. **`WA_Audit_Oversight_Findings.pdf`** (Section E/F)
   - Extraction of WA SAO audit findings on HCA's oversight processes, compared structurally to findings from Minnesota's legislative auditor.
6. **`WA_MN_Comparison_Matrix.pdf`** (Section G)
   - Structural comparison of Washington's licensing model against Minnesota's; see the memo's status note for how this framing was revised.
7. **`archive/WA_Public_Data_Review_Priority_Tier.pdf`** (Section H, ARCHIVED)
   - A never-adopted, illustrative 0-100 point rubric, moved to `archive/` and given a prominent warning banner because a numeric rubric like this is easy to misread out of context as a real risk score. The manuscript's Appendix B qualitative PDRT framing is canonical; this memo is not used anywhere in the manuscript, dataset, or scripts.
8. **`WA_Data_Availability_Matrix_Updated.pdf`** (Section I)
   - Explicit accounting of what is public vs. what requires a Public Records Request (PRR).
9. **`WA_Readiness_Memo.pdf`** (Section J)
   - Executive synthesis of the pilot's findings, limitations, and recommendations for deployment.
10. **`WA_Expanded_Bibliography.pdf`** (Section F)
    - Comprehensive list of all primary sources used in the extended pilot.

*(All markdown documents have been converted to PDF for easy distribution).*
