# Data Availability Matrix: Minnesota Medicaid Fraud Research

This matrix details the public availability of the data types requested for this evidence base.

| Data Category | Availability | Source / Methodology | Caveats / Limitations |
| :--- | :--- | :--- | :--- |
| **Enforcement Cases (Federal)** | Public | DOJ Press Releases, Criminal Division Case Summaries | Represents only charged/alleged crimes; does not include sealed indictments. |
| **Enforcement Cases (State)** | Public | MN AG/MFCU Press Releases | Represents only charged/alleged crimes. |
| **Program Spending Growth** | Public (Partial) | Extracted from DOJ charging documents and DHS fact sheets | Comprehensive year-by-year claims data requires a public records request to DHS; figures used are those cited in indictments. |
| **Revalidation Outcomes** | Public | DHS Program Integrity Page, DHS Fact Checks | Provides aggregate totals (e.g., 3,411 disenrolled); does not name the specific providers disenrolled for administrative reasons. |
| **Audit Findings** | Public | MN Office of the Legislative Auditor (OLA), HHS-OIG, CMS | Full unredacted third-party (Optum) data analytics reports are not public (currently subject to legislative debate). |
| **Ownership/Network Data** | Partial | MN Secretary of State Business Search, CMS NPPES Registry | SOS shows registered agents and managers; NPPES shows NPI data. However, true beneficial ownership and comprehensive related-party networks (42 CFR 455.104 disclosures) are held by DHS and are not fully public without a data request. |
| **Complaint/Tip Data** | Not Public | KARE 11 Investigative Reporting | Raw county fraud referrals and internal DHS emails are not public; they were obtained by journalists via data practices requests. |
